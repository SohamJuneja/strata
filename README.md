# Strata — Structured Options Vaults on Sui

**Earn yield from DeepBook Predict's binary options market with built-in crash protection.**

Strata packages DeFi options exposure into structured vaults that any LP can use without understanding options math. Two strategies, AI-powered vault creation, and real on-chain activity generated by a continuous integration bot across 5 wallets.

Built for Sui Overflow 2026 — DeepBook Track.

---

## The Problem

Raw LP exposure on options protocols is brutal in crashes: you're on the wrong side of every payout. On Ethereum, Ribbon Finance and Friktion hit $1B TVL with structured vaults — but both are now shut down because options were underpriced. Strata solves this on Sui using DeepBook Predict's live vol surface pricing, so the premium is always fair, and adds built-in hedging to protect LPs when BTC moves hard.

---

## How It Works

Strata vaults accept dUSDC deposits and mint `VAULT_SHARE` tokens (standard Sui Coin) to depositors. Those shares are composable — stackable with margin and lending in a single PTB. Under the hood, the vault allocates capital across two sources of yield and one risk buffer:

**STRATA-PH (PLP + Hedge)**
- 90% of deposits supply liquidity to DeepBook Predict's binary options market, earning premium as options settle
- 10% buys OTM binary put options as crash insurance — when BTC drops past the strike, the hedge fires and pays out, offsetting LP losses
- The hedge is evaluated and redeemed permissionlessly every oracle period (15 min)

**STRATA-RL (Range Ladder)**
- Five range positions around the current ATM BTC price
- Direction-neutral: pays out if BTC lands anywhere within the ladder on settlement
- For LPs without a directional view

## AI Vault Creator

Anyone can launch their own strategy vault using plain English:
1. Describe your strategy in a chat interface
2. GPT-4o-mini extracts strategy parameters (hedge ratio, strike offset, risk level)
3. One click deploys the config to Sui testnet via `strata::vault_factory::register_vault`
4. Community vaults are registered on-chain and appear in the UI with a "Verified on-chain" badge
5. Vault creator earns 0.10% fee on all deposits

Try it at `/create-vault`.

## Architecture

```
┌─────────────────────────────────────────┐
│         DeepBook Predict                 │
│  (Binary options, 15-min oracle, BTC)    │
└──────────────┬────────────────────────────┘
               │ deploy_to_predict / buy_put
┌──────────────▼────────────────────────────┐
│           Strata Vault                    │
│  vault.move + strategy_plp_hedge.move     │
│  strategy_range_ladder.move                │
│  vault_factory.move (community vaults)    │
└──────────────┬────────────────────────────┘
               │ VAULT_SHARE (Coin<VAULT_SHARE>)
┌──────────────▼────────────────────────────┐
│         LP / DeFi ecosystem               │
│  (margin, lending, composable in PTBs)    │
└─────────────────────────────────────────┘
```

**Move contracts** (`sources/`): `vault.move`, `vault_share.move`, `strategy_plp_hedge.move`, `strategy_range_ladder.move`, `vault_factory.move`

**Frontend** (`frontend/`): Next.js 16, Sui dApp Kit, Tailwind CSS — vault pages, AI creator, vol dashboard, activity feed

**Marathon bot** (`marathon-bot/`): TypeScript, 5 LP wallets, 15-min cycles, continuous integration testing

**Keeper bot** (`keeper/`): Permissionless hedge settlement

## Running Locally

```bash
git clone https://github.com/SohamJuneja/strata
cd strata/frontend
npm install
cp .env.local.example .env.local
# Add OPENAI_API_KEY and OPERATOR_PRIVATE_KEY to .env.local
npm run dev
# Open http://localhost:3000
```

For the marathon bot:
```bash
cd marathon-bot
npm install
cp .env.example .env
# Fill in addresses from the table above
npm start
```
